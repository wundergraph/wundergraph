# WunderGraph cross API joins example

#### Getting started

```shell
npm install && npm start
```

#### Get weather for a country capital

```shell
curl -v --get --data-urlencode 'wg_variables={"countryCode": "ES"}' 'http://localhost:9991/operations/CountryWeather'
```
