# WunderGraph Node.js + TypeScript Client

This is a simple example of how to use the WunderGraph auto-generated and type-safe client with Node.js and TypeScript.

#### Getting started

```shell
npm install && npm start
```

#### Run the example

```shell
npm run request
```

#### Tests

```shell
npm test
```
