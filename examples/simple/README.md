# WunderGraph simple example

#### Getting started

```shell
npm i && npm start
```

#### Get all Continents

```shell
curl http://localhost:9991/operations/Continents
```

#### Get all Countries

```shell
curl http://localhost:9991/operations/Countries
```

#### Get user

```shell
curl http://localhost:9991/operations/users/get?id=1
```

## Test the API

1. (Optional) Copy the `.env.example` file to `.env.test` and fill in the required values.
2. Run the following command:

```shell
npm run test
```
